# Ubuntu
## 开启root用户登录桌面
```
sudo vim /usr/share/lightdm/lightdm.conf.d/50-ubuntu.conf
加上下面一句
greeter-show-manual-login=true
然后修改密码
sudo passwd root
重启系统```

---
## 允许root用户登录ssh
```
/etc/ssh/sshd_config
修改PermitRootLogin
PermitRootLogin yes
#保存退出重启服务
sudo service ssh restart
```
